PDFILEdit

PDFILEdit is a Flask-based web application developed as my CS50x final project to tackle the common frustrations of PDF editing faced by students and professionals. It offers a clean, ad-free, browser-based solution that supports multiple PDF tasks in one place—making it easy to merge, sign, extract, convert, and compress PDFs without switching tools. Users can upload PDF files which are stored in secure, session-specific directories to ensure privacy and scalability. The system dynamically adapts to user input: single PDFs enable functions like signing, page extraction, and conversion, while multiple uploads activate the merge and ZIP features. All actions are integrated into a smooth UI/UX that preserves file visibility across different app routes.

Technical Highlights:

  • Backend: Flask, Python, PyPDF2
  • Frontend: HTML5, JavaScript (Canvas), CSS
  • File Handling: Secure uploads with secure_filename; session-based isolation for concurrent users
  • Signature Feature: HTML5 Canvas for drawing, resizing, and dragging signatures over rendered PDFs
  • Dynamic Routing: Files persist across pages for seamless switching between tools
  • Storage Management: Hybrid cleanup via user-facing buttons and server-side time-based deletion
  • PDF Editing Features: PDF merging, single-page extraction, format conversion, ZIP compression of multiple PDFs
This project reflects my focus on building intuitive, real-world tools with attention to scalability, user experience, and technical precision.

CS50 Finance

Built using Flask, Python, and SQL (with SQLite for local development and PostgreSQL in production), the app allows users to:

  1. Create an account and log in securely
  2. Search for stock quotes using the CS50 Finance API
  3. Buy and sell stocks, which are then stored in the user’s transaction history
  4. Display portfolio information, including total value and individual stock holdings
  • Authentication & Security: Implemented user authentication using Flask's session management and secured user passwords with Flask-Bcrypt.
  • Database Management: Used SQLAlchemy and PostgreSQL to store user data, transactions, and portfolio details securely in the cloud.
  • Version Control & Deployment: Managed the project using Git and deployed the final product with Render, integrating the backend with a real PostgreSQL database hosted on Render.
